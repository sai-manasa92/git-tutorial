## GIT Assignment-1
- Setup a github account
- Setup SSH Keys for your working computer with your account
- Create a repository in github
- Invite your peers to join the repository
- Commit a README.md file to the repository and highlight the things that you learned in git
- Create a branch in that repo, and create a pull request
- Request a review by one of the peers you invited into your repo
- Once they approve merge in the code into the master branch
